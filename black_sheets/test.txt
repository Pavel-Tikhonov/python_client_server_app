1. Создайте html-документ с тегом script, внутри которого определите три переменные с разными типами данных:
- переменная num со значением 18
- переменная str со значением '81'
- переменная isStr со значением true

Добавьте переменную sum и присвойте ей сумму переменных num и str.
Выведите в консоль браузера через команду console.log() информацию о всех переменных, используя оператор typeof. Пример: console.log("typeof num", typeof num, num).

2. Создайте html-документ и подключите к нему внешний js-файл. Содержимым js-файла должны быть переменные со значениями:
- -0
- -1
- ""
- "Привет"
- null
- []
Каждую переменную выведите в консоль браузера, преобразовывая значение в булевый тип.
Пример:
const str1 = 12;
console.log('str1', Boolean(str1));

3. Создайте html-документ и подключите к нему внешний js-файл. Содержимым js-файла должна быть переменная со строкой "Привет, программистам!".
Выведите в консоль браузера переменную следующими способами:
- вывод строки всеми большими буквами
- вывод строки всеми маленькими буквами
- вывод только первого символа строки
- вывод только последнего символа строки
